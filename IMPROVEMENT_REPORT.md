#### B. 菜单布局改为 2x3 网格 ⭐

**之前**：纵列排列（占据太多空间）
```
[攻击]
[技能]
[道具]
[切换精灵]
[逃跑/认输]
```

**改进后**：2x3 网格布局
```
┌─────────────────────────────────────┐
│  [攻击]   [技能]   [捕提]         │
│  [道具]   [切换]   [逃跑/认输]   │
└─────────────────────────────────────┘
```

**代码改进**：
```javascript
// 新的网格菜单
const buttonW = (SCREEN_WIDTH - 48) / 3;  // 三列均分
const buttonH = 48;
const gapX = 8;
const gapY = 8;

// 第一行
- [攻击] 红色 X=16
- [技能] 黄色 X=16 + buttonW + gapX
- [捕提] 青色 X=16 + (buttonW + gapX)*2

// 第二行 (Y += 56)
- [道具] 绿色 X=16
- [切换] 青色 X=16 + buttonW + gapX
- [逃跑/认输] 青色/紫蓝色 X=16 + (buttonW + gapX)*2
```

**效果**：
- 菜单更紧凑，屏幕利用率提高
- 按钮大小更均匀
- 易于触摸操作（更宽敞）
- 留出底部区域显示更多日志
